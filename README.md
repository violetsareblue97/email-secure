Email Safe+ (Email Phishing Detector)

Aplikasi berbasis web untuk mendeteksi email phishing menggunakan Machine Learning dan verifikasi DNS.

Proyek ini menggunakan dataset publik dari Kaggle untuk melatih model NLP.
Dataset Source: https://www.kaggle.com/datasets/naserabdullahalam/phishing-email-dataset

Cara kerja sistem:
1. Analisis Teks
   Menggunakan NLP untuk memeriksa pola manipulasi pada badan email.
3. Verifikasi Identitas
   Mengecek protokol SPF/DMARC.

KUNJUNGI WEBSITE: https://emailsecure.streamlit.app/
